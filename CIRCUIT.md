## Connections

- Arduino (Master) to SPI Slave: `MOSI`, `MISO`, `SCK`, `SS`. Common `GND`.

## Diagram (ASCII)

```
 Arduino MOSI ---- MOSI Slave
 Arduino MISO ---- MISO Slave
 Arduino SCK  ---- SCK  Slave
 Arduino SS   ---- CS   Slave
 GND          ---- GND  Slave
```
